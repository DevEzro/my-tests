# USO DE TESTCAFE
## INSTALACIÓN
- Instalar node.js
- Abrir node y comprobar versiones: `node -v` `npm -v`
  ![image](https://github.com/user-attachments/assets/7958d34f-f98e-49c5-bb59-129709889a75)

## SETUP
- Instalar TestCafé
`npm install testcafe --save-dev`
- Ejecutar pruebas
`npx testcafe chrome </ruta/al/fichero.js>`
